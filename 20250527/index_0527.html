CSS 속성 (Attribute)

레이아웃 관련 속성

width, height

태그의 가로/세로 크기 지정

  • 작성법: width: 크기; height: 크기;
  • span은 기본적으로 inline이므로 크기 적용되지 않음
  • div는 block 요소이므로 영역 확보 가능

display

태그의 영역 방식 정의

  • inline, block, inline-block, none
  • inline-block은 공백 조심 (문자 간격 때문)

padding, margin

태그 내부/외부 여백 지정

  • padding: 내부 여백, margin: 외부 여백
  • 방향 지정 가능: padding-left, margin-top 등
  • margin: auto;는 좌우 자동 여백 → 중앙 정렬에 사용

border, outline

테두리 설정

  • 작성법: border: 종류 두께 색상;
  • 종류: dashed, solid 등
  • none으로 제거 가능

box-sizing

크기 계산 기준 설정

  • content-box: padding은 크기 외로 계산
  • border-box: padding, border 포함 크기로 계산

background-color

배경색 지정

background-image

배경 이미지 또는 그라디언트 지정

  • url('경로') 방식
  • linear-gradient, radial-gradient 가능

background-size

배경 이미지 크기 설정

  • cover: 큰 축 맞춤
  • contain: 작은 축 맞춤

border-radius

테두리 둥글기 설정

overflow

내용이 넘칠 때 처리 방식

  • visible, hidden, scroll

object-fit

미디어(이미지/비디오 등) 크기 조절 방식

  • cover, contain, fill, none, scale-down

left, right, top, bottom

위치 이동

  • 해당 방향 기준선에서 반대 방향으로 이동
  • 예: left: 50px → 왼쪽 기준선에서 오른쪽으로 50px 이동

position

기준선 위치 지정

  • static: 기본
  • relative: 현재 위치 기준
  • absolute: 상위 기준선 기준
  • fixed: 브라우저 화면 기준
  • sticky: 특정 위치 도달 시 고정

visibility, opacity, z-index

가시성 제어

  • visibility: visible / hidden
  • opacity: 0.0 ~ 1.0 (투명도)
  • z-index: 앞뒤 우선순위 설정

transform

요소 변형

  • rotate(각도), translate(x,y), scale(비율), perspective(원근)

float

요소를 좌/우로 띄움

  • left, right

clear

float 해제

  • left, right, both

cursor

마우스 포인터 형태 지정

user-select

텍스트 선택 가능 여부 지정

box-shadow, text-shadow

그림자 생성

  • 작성법: box-shadow: 거리 색상 크기;

filter

이미지 효과 적용

  • blur, grayscale, sepia, contrast 등
