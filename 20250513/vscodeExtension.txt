# 설치하면 편리한 모드

1. Auto Close Tag
    태그 작성시 열림 태그를 적으면 닫힘 태그가 자동 생성
    <tag></tag>

2. Auto Rename tag
    태그 수정시 열림/닫힘 태그 둘 중 한개만 바꿔도 나머지도 바뀌도록 하는 기능

3. Color Picker
    코드에 적히는 색상을 직접 보면서 수정할 수 있도록 만들어주는 기능

4. Indent-rainbow
    탭에 색상 부여

5. Material Icon Theme
    파일의 이름을 기반으로 아이콘 변경
