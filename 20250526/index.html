CSS 속성 (Attribute)

텍스트 관련 속성

font-family

폰트를 설정하는 속성

  • 작성법: font-family: 폰트이름, 폰트이름;
  • PC에 설치된 폰트만 사용 가능
  • 여러 폰트를 순서대로 지정 가능 (앞 폰트가 없을 시 뒤 폰트 사용)
  • 영문/한글/기타 언어 순으로 지정하는 것이 일반적

@font-face

외부 폰트를 불러오는 기능 (속성 아님)

@font-face {
        font-family: '폰트이름';
        src: url(폰트URL);
        }
  • 외부 폰트를 자동 다운로드하여 적용
  • 폰트 저작권 확인 필수

font-size

글자 크기를 지정하는 속성

브라우저 기본값: 대부분의 브라우저 (Chrome, Firefox, Edge, IE 포함)에서 기본 html 요소의 font-size는 16px(1rem = 16px 기준으로 배수 계산)

  • px: 고정값
  • em: 상위 요소 기준 배율
  • rem: html 기준 배율
  • %: em과 유사 (비율)
  • vw: 뷰포트 너비 기준
  • vh: 뷰포트 높이 기준

font-weight

글자 굵기 지정

  • 숫자: 100~900
  • 문자: lighter, bold, bolder

font-style

글자 스타일(기울기) 지정

  • normal, italic, oblique

text-decoration

문장에 줄을 긋는 속성

  • none: 없음
  • underline: 밑줄
  • overline: 윗줄
  • line-through: 취소선

color

글자 색상 지정

  • 색상 이름: red, blue, green 등
  • 16진수: #fff00f 등
  • rgb: rgb(255, 255, 0)
  • rgba: rgba(255, 255, 0, 0.5)
  • hsl: hsl(255, 255, 0)
  • oklch: oklch(% 0.1 150)

text-align

글자 정렬

  • left, center, right

line-height

줄 간격 설정

word-spacing

단어 사이 간격 설정

letter-spacing

글자 사이 간격 설정

text-indent

문장 첫 줄 들여쓰기

white-space

줄바꿈 제어

  • normal: 기본 줄바꿈 허용
  • nowrap: 줄바꿈 제거

text-overflow

줄바꿈이 없을 때 말줄임 처리

  • clip: 자르기
  • ellipsis: 말줄임표 (...)
  • 주의: white-space: nowrap 및 overflow: hidden 조건 필요

line-break

어떤 기준으로 줄을 나눌지 결정

  • normal, auto: 띄어쓰기 단위로 줄바꿈
  • loose: 띄어쓰기 외의 위치에서도 줄바꿈 가능
  • anywhere: 어디서든 줄바꿈 허용
