기본 작성 형식

        Selector {
        Attribute-Name: Value;
        Attribute-Name: Value;
        }

CSS Selector 종류

기본 선택자

  • Tag 선택자: div {}
  • Class 선택자: .class {}
  • Id 선택자: #id {}
  • 전체 선택자: * {}

관계 선택자

  • AND 선택자: div.red 또는 .a#b
  • Have 관계: .a #b
  • Have 직접 관계: .a > #b
  • Sibing 관계: .a + .b
  • Sibing 전체 관계: .a ~ .b
  • OR 선택자: .a, .b
  • 속성 선택자:
    • [type]
    • [type="password"]
    • [value="test"]

특수 선택자 (가상 선택자)

  • nth 관련:
    • :first-child, :last-child
    • :nth-child(n), :nth-last-child(n)
    • :first-of-type, :last-of-type
    • :nth-of-type(n), :nth-last-of-type(n)
  • :active
  • :hover
  • :focus
  • :visited, :link
  • :has(selector)
  • :where(a, b)
  • :is(a, b)
  • :not(selector)

주의: Inline 방식은 항상 최우선. 동일한 방식 내에서는 나중에 선언된 것이 우선됨
