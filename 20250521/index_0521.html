Request 태그

1. form - 서버에 정보 요청

form 태그는 내부 input 등의 입력 값을 서버로 전송하는 데 사용

주요 속성

  • action: 데이터를 보낼 대상 주소
  • method: 데이터를 보내는 방식 (GET / POST)
  • enctype: 데이터 인코딩 방식

action 예시

  • https://www.naver.com/path/to (Protocol + Domain 포함 → 다른 서버로 요청)
  • /path/to (현재 웹페이지와 같은 서버에 요청)

method 예시

  • GET: 주소에 데이터를 포함 (예: ?a=b&c=d)
  • POST: 데이터를 Body 내부에 포함 (주소에 노출되지 않음)

enctype 예시

  • application/x-www-form-urlencoded: 기본값, 텍스트 전송
  • multipart/form-data: 파일 전송 시 사용
