Animation 속성

간단 Animation: transition


        transition: background-color 1s ease 0s;
        transition: width 0.5s linear 0s;
        transition: all 0.3s ease-in-out 0s;
  • transition: 대상 시간 공식 딜레이;
  • 시간 단위 예시:
    • 1s
    • 0.5s
  • 공식 예시:
    • linear
    • ease-in
    • ease-out
    • ease-in-out
    • cubic-bezier()
  • 딜레이: 0s
  • 주의: 숫자가 아닌 속성(예: font-name)은 서서히 변화 불가

복잡 Animation: keyframes + animation

@keyframes


        @keyframes move {
        0%   { left: 0px; }
        50%  { left: 250px; }
        100% { left: 0px; }
        }

animation


        animation: move 3s ease-in-out 0s alternate infinite running;
  • animation: 이름 시간 공식 딜레이 방향 횟수 상태
  • 방향:
    • normal
    • alternate
    • reverse
    • alternate-reverse
  • 횟수 예시:
    • 1
    • infinite
  • 상태:
    • running
    • paused
