CSS 확장 기능: 변수, 덮어쓰기, 반응형

CSS 변수

:root {
    --brand-color: hsl(182, 83%, 43%);
    }

    h1 {
    color: var(--brand-color);
    }

--brand-color는 페이지 전체에서 사용할 수 있는 변수이며, var() 함수로 값을 불러올 수 있다.

덮어쓰기

!important

h1 {
    color: red !important;
    }

강제적으로 해당 스타일을 적용

@layer

@layer base, theme, component;

    @layer base {
    h1 { color: red; }
    }
    @layer component {
    h1 { color: blue; }
    }

@layer는 CSS 렌더링 우선순위를 명시적으로 조절할 수 있게 해준다.

반응형 웹: Media Query

@media screen and (max-width: 600px) {
    h1 {
        font-size: 1.2rem;
    }
    }

브라우저 화면 너비가 600px 이하일 때 적용되는 조건부 스타일

Container Query

div {
    container-type: inline-size;
    container-name: exampleContainer;
    }

    @container exampleContainer (min-width: 500px) {
    h1 {
        color: green;
    }
    }

해당 div 내부 크기가 500px 이상일 경우에만 h1 색상을 green으로 변경

단위

  • cqw: container width의 1%
  • cqh: container height의 1%
  • cqmin: width, height 중 작은 값의 1%
  • cqmax: width, height 중 큰 값의 1%
