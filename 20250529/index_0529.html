레이아웃 속성: Flex & Grid

Flex 예시

Flex: 유연한 레이아웃 배치를 위한 속성

/* .row: Flex container 설정 */
        .row {
        width: 150px;
        height: 150px;
        background-color: greenyellow;
        display: flex; /* flex 레이아웃 지정 */
        flex-direction: row; /* 주 축 방향: 가로 */
        flex-wrap: nowrap;   /* 넘침 방지 없음 */
        }

        /* .item: Flex item 설정 */
        .item {
        border: 1px black solid;
        /* flex-grow: 1; */
        /* flex-shrink: 0; */
        }

Flex 주요 속성

  • display: flex
  • flex-direction: row
  • flex-wrap: nowrap
  • justify-content: center
  • align-items: center

Grid 예시

Grid: 2차원 격자 배치를 위한 속성

/* .container: Grid container 설정 */
        .container {
        width: 50vw;
        height: 150px;
        background-color: yellow;
        display: grid; /* grid 레이아웃 지정 */
        grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
        }

        /* .id_01: span을 활용한 행 확장 */
        .id_01 {
        grid-row-end: span 3;
        }

Grid 주요 속성

  • display: grid
  • grid-template-columns: repeat(auto-fill, minmax(150px, 1fr))
  • grid-row-end: span 3

추가: Emmet 문법

HTML 구조를 빠르게 생성할 수 있는 단축 문법, VSCode 등에서 지원

/* div.container>div.item.id_${$}*9 */
  • div.container: class가 container인 div 생성
  • >: 자식 요소로 연결
  • div.item.id_${$}: item 클래스와 id_01 ~ id_09까지 자동 생성
  • *9: 위 구조를 9번 반복

위 Emmet 구문을 입력 후 Tab을 누르면 자동 완성
